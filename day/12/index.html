MultiIndex: Hierarchical Indexing

A MultiIndex allows you to have multiple index levels on a DataFrame or Series. Perfect for representing higher-dimensional data in a 2D structure.

import pandas as pd

# Create MultiIndex from tuples
arrays = [['A', 'A', 'B', 'B'], [1, 2, 1, 2]]
index = pd.MultiIndex.from_arrays(arrays, names=['letter', 'number'])
df = pd.DataFrame({'value': [10, 20, 30, 40]}, index=index)

print(df)
#              value
# letter number       
# A      1         10
#        2         20
# B      1         30
#        2         40

MultiIndex Operations:

df.xs('A')→ cross-section at level
df.swaplevel()→ swap index levels
df.unstack()→ pivot inner level to columns
df.stack()→ pivot columns to index

Pivot Tables & Melt: Reshaping Data

pivot_table() - Wide Format

Create a spreadsheet-style pivot table from a DataFrame.

pd.pivot_table(df, values='sales', 
               index='region', 
               columns='product', 
               aggfunc='sum')

melt() - Long Format

Unpivot a DataFrame from wide to long format.

pd.melt(df, id_vars=['id'], 
        value_vars=['col1','col2'],
        var_name='variable',
        value_name='value')

📐 pivot_table = wide (spreadsheet-like) | melt = long (tidy data for analysis)

Time Series: Dates, Resampling, Rolling Windows

import pandas as pd

# Create date range
dates = pd.date_range('2024-01-01', periods=100, freq='D')
df = pd.DataFrame({'value': range(100)}, index=dates)

# Resample to monthly
monthly = df.resample('M').mean()

# Rolling window (7-day average)
rolling = df['value'].rolling(window=7).mean()

# Shift and lag
df['prev_day'] = df['value'].shift(1)

📅 Time Series Features:

• date_range() - Generate datetime indices
• resample() - Change frequency (D, W, M, Q, Y)
• rolling() - Moving window calculations
• shift() - Lag/lead values
• dt accessor - Extract year/month/day
• tz_localize/tz_convert - Timezone handling

📈 Example: 7-day rolling average smooths out daily fluctuations to reveal trends.

Categorical Data: Memory Optimization

Convert object columns to category dtype to save memory and speed up operations when you have repeated values.

# Convert to categorical
df['department'] = df['department'].astype('category')

# Check memory usage
df.memory_usage(deep=True)

# Category codes (internal integer representation)
df['dept_code'] = df['department'].cat.codes

📊 Memory Savings Example:

• 10 million rows, 'status' column with 5 unique values
• Object dtype: ~80 bytes per string → 800 MB
• Category dtype: ~1 byte per integer + 5 strings → ~10 MB
✓ 98% memory reduction!

Performance Optimizations: Do's & Don'ts

✅ DO (Fast)

  • Use vectorized operations (df[col] * 2)
  • Use .loc/.iloc instead of chained indexing
  • Use categorical dtypes for repeated strings
  • Use query() for complex filtering
  • Use eval() for arithmetic on large DataFrames
  • Read only necessary columns with usecols

❌ DON'T (Slow)

  • Iterating rows with iterrows() or apply() with lambda
  • Growing DataFrame in a loop (append inside loop)
  • Chained indexing (df['col']['row'])
  • Using python string methods instead of .str
  • Converting from DataFrame to list and back

import pandas as pd
import numpy as np

# ========== 1. MULTIINDEX ==========
arrays = [['North', 'North', 'South', 'South'], 
          ['Q1', 'Q2', 'Q1', 'Q2']]
index = pd.MultiIndex.from_arrays(arrays, names=['Region', 'Quarter'])
sales_df = pd.DataFrame({'Sales': [100, 120, 90, 110], 'Profit': [20, 25, 18, 22]}, index=index)
print("MultiIndex DataFrame:\n", sales_df)

# ========== 2. PIVOT TABLE ==========
df = pd.DataFrame({
    'Date': pd.date_range('2024-01-01', periods=100),
    'Product': np.random.choice(['A', 'B'], 100),
    'Sales': np.random.randint(100, 500, 100)
})
pivot = pd.pivot_table(df, values='Sales', index=df['Date'].dt.month,
                       columns='Product', aggfunc='sum')

# ========== 3. TIME SERIES ==========
ts = pd.Series(np.random.randn(1000), index=pd.date_range('2024-01-01', periods=1000))
rolling_mean = ts.rolling(window=30).mean()
monthly = ts.resample('M').mean()

# ========== 4. STRING OPERATIONS ==========
text_df = pd.DataFrame({'text': ['  Hello World  ', 'Python,Pandas,NumPy', 'DataScience']})
text_df['clean'] = text_df['text'].str.strip().str.lower()
text_df['split'] = text_df['text'].str.split(',')

# ========== 5. CATEGORICAL (Memory Optimization) ==========
large_df = pd.DataFrame({'category': np.random.choice(['low', 'medium', 'high'], 10000)})
large_df['category'] = large_df['category'].astype('category')
print(f"Memory: {large_df.memory_usage(deep=True)[0]} bytes")
📊 Output: MultiIndex works | Pivot table created | Rolling window applied | String ops vectorized | Memory optimized with categorical!
