Filter Methods: Statistical Feature Ranking

Filter methods rank features independently of any machine learning model using statistical tests.

📊 Variance Threshold:

Removes features with variance below threshold (constant or quasi-constant features).

selector = VarianceThreshold(threshold=0.01)
📈 Pearson Correlation:

Measures linear relationship with target. Good for linear models.

corr = df.corr()['target'].abs().sort_values(ascending=False)
🎯 ANOVA F-test:

Compares means across groups. Best for categorical target, continuous features.

selector = SelectKBest(score_func=f_classif, k=10)
💡 Mutual Information:

Captures any relationship (linear/non-linear). Most powerful filter method.

mi = mutual_info_classif(X, y)

✅ Advantages of Filter Methods:

  • Very fast and scalable to high dimensions
  • Independent of model choice
  • Excellent for initial feature screening
  • Good statistical foundation

⚠️ Limitation: Ignores feature interactions and model bias

Wrapper Methods: Model-Based Selection

🔄 Recursive Feature Elimination (RFE):

Recursively removes least important features based on model coefficients.

rfe = RFE(estimator=LogisticRegression(), n_features_to_select=5)
⬆️ Forward Selection:

Start with empty set, iteratively add best performing feature.

⬇️ Backward Elimination:

Start with all features, iteratively remove worst performing.

🔁 Bidirectional (Stepwise):

Combine forward and backward selection.

✅ Advantages of Wrapper Methods:

  • Model-aware — finds features optimized for specific algorithm
  • Considers feature interactions
  • Often yields better performance than filters

⚠️ Limitation: Computationally expensive (O(n²) to O(2ⁿ))

Embedded Methods: Built-in Feature Selection

📉 Lasso Regression (L1):

Pushes coefficients to zero → automatic feature selection.

from sklearn.linear_model import Lasso
🌲 Tree-based Importance:

Random Forest, XGBoost, LightGBM provide feature importance scores.

rf.feature_importances_
⚡ Elastic Net:

Combines L1 (Lasso) and L2 (Ridge) regularization.

from sklearn.linear_model import ElasticNet

✅ Advantages of Embedded Methods:

  • Computationally efficient (between filter and wrapper)
  • Model-aware with feature interactions
  • Built-in regularization prevents overfitting

💡 Best for: High-dimensional data, when you need fast and reliable selection

Hybrid & Advanced Techniques

Boruta Algorithm

Compares original features with randomized shadow features to determine importance.

from boruta import BorutaPy

Permutation Importance

Measures drop in model performance when a feature's values are randomly shuffled.

from sklearn.inspection import permutation_importance

Auto-Feature Selection

Automated selection using validation curves and cross-validation.

from sklearn.feature_selection import SelectFromModel

Boruta: Feature is important if its Z-score > max(Z_score of shadow features)

💡 Practical Feature Selection Workflow

1. Remove constant/quasi-constant features (Variance Threshold)
2. Remove highly correlated features (correlation > 0.95)
3. Apply Mutual Information/ANOVA to rank remaining
4. Use RFE or SelectFromModel for final selection
5. Validate with cross-validation and permutation importance
6. Iterate and refine based on model performance

Complete Implementation: All Methods in Action

advanced_feature_selection.py 
import pandas as pd
import numpy as np
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.ensemble import RandomForestClassifier
from sklearn.feature_selection import (
    VarianceThreshold, SelectKBest, f_classif, mutual_info_classif,
    RFE, SelectFromModel, RFECV
)
from sklearn.linear_model import LassoCV, LogisticRegression
from sklearn.preprocessing import StandardScaler

# Load dataset
data = load_breast_cancer()
X = pd.DataFrame(data.data, columns=data.feature_names)
y = data.target

# Scale features (important for Lasso, RFE)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

print(f"Original shape: {X.shape}")

# ========== 1. FILTER METHODS ==========
# Variance Threshold (remove constant features)
selector_var = VarianceThreshold(threshold=0.01)
X_var = selector_var.fit_transform(X)
print(f"After Variance Threshold: {X_var.shape[1]} features")

# Mutual Information (best filter)
mi_scores = mutual_info_classif(X, y)
mi_selected = np.argsort(mi_scores)[-10:]  # Top 10

# ========== 2. WRAPPER METHODS ==========
# RFECV (Recursive Feature Elimination with CV - automatic k selection)
estimator = LogisticRegression(max_iter=1000, random_state=42)
selector_rfecv = RFECV(estimator, step=1, cv=5, scoring='accuracy')
selector_rfecv.fit(X_scaled, y)
print(f"RFECV optimal features: {selector_rfecv.n_features_}")

# ========== 3. EMBEDDED METHODS ==========
# Lasso (L1 regularization)
lasso = LassoCV(cv=5, random_state=42, max_iter=10000)
lasso.fit(X_scaled, y)
lasso_selected = np.where(lasso.coef_ != 0)[0]
print(f"Lasso selected: {len(lasso_selected)} features")

# Random Forest Importance
rf = RandomForestClassifier(n_estimators=100, random_state=42)
rf.fit(X, y)
importance_df = pd.DataFrame({
    'feature': X.columns,
    'importance': rf.feature_importances_
}).sort_values('importance', ascending=False)
rf_selected = importance_df.head(10)['feature'].tolist()

# SelectFromModel (automated threshold)
selector_sfm = SelectFromModel(RandomForestClassifier(n_estimators=100), threshold='median')
selector_sfm.fit(X, y)
X_sfm = selector_sfm.transform(X)
print(f"SelectFromModel selected: {X_sfm.shape[1]} features")

# ========== 4. COMPARE PERFORMANCE ==========
X_train, X_test, y_train, y_test = train_test_split(X, y, random_state=42)

# Baseline: All features
rf_full = RandomForestClassifier(random_state=42)
scores_full = cross_val_score(rf_full, X_train, y_train, cv=5)

# Using top RF features
X_train_selected = X_train[rf_selected]
rf_selected_model = RandomForestClassifier(random_state=42)
scores_selected = cross_val_score(rf_selected_model, X_train_selected, y_train, cv=5)

print(f"All features CV score: {scores_full.mean():.4f}")
print(f"Selected features CV score: {scores_selected.mean():.4f}")
print(f"Top 10 features: {rf_selected[:5]}...")
📊 Output: Original: 30 features → RFECV: 12 optimal → Lasso: 8 features → Accuracy maintained/improved!

Decision Guide: Which Method Should You Use?

📊 Use FILTER when:

  • You have hundreds/thousands of features
  • Need quick initial screening
  • Features are independent
  • Computational resources are limited

🎯 Use WRAPPER when:

  • Feature set is small (<50 features)
  • Maximum predictive performance is critical
  • Feature interactions are important
  • You have a specific model in mind

⚡ Use EMBEDDED when:

  • You need a balance of speed and accuracy
  • Using tree-based models or regularized linear models
  • Feature set is medium (50-500 features)
  • Want built-in regularization

🔬 Use HYBRID when:

  • Research/competition setting
  • Need maximum robustness
  • Can afford computational cost
  • Want to validate with multiple methods

⚠️ Common Pitfalls in Feature Selection

→ Selecting features on entire dataset (data leakage!)
→ Using only one method without validation
→ Ignoring domain knowledge
→ Not scaling features for Lasso/RFE
→ Over-optimizing feature count
→ Forgetting to apply selection to test data
