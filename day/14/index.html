Why Feature Selection Matters

🚀 Faster Training

Fewer features = less computation time and memory usage.

📈 Better Accuracy

Removing noise and irrelevant features reduces overfitting.

🔍 Interpretability

Simpler models are easier to explain and debug.

📌 The Curse of Dimensionality: As the number of features increases, data becomes sparse and models require exponentially more samples. Feature selection is crucial!

Feature Selection Methods

Filter Methods

  • Variance Threshold
  • Correlation-based (Pearson)
  • Chi-Square Test
  • Mutual Information

Fast, independent of model

Wrapper Methods

  • Forward Selection
  • Backward Elimination
  • Recursive Feature Elimination (RFE)
  • Exhaustive Search

Model-specific, more accurate but slower

Embedded Methods

  • Lasso Regression (L1)
  • Ridge Regression (L2)
  • Decision Tree Importance
  • Random Forest Importance

Built into model training

Complete Python Implementation

feature_selection_demo.py 
import pandas as pd
import numpy as np
from sklearn.feature_selection import VarianceThreshold, SelectKBest, f_classif, RFE
from sklearn.ensemble import RandomForestClassifier
from sklearn.preprocessing import StandardScaler

# ========== 1. CREATE SAMPLE DATASET ==========
np.random.seed(42)
df = pd.DataFrame({
    'feature_1': np.random.randn(100),
    'feature_2': np.random.randn(100) * 2,
    'feature_3': np.random.randn(100) + 5,
    'feature_4': np.random.randn(100) * 0.1,  # Low variance
    'target': np.random.choice([0, 1], 100)
})

X = df.drop('target', axis=1)
y = df['target']

# ========== 2. CORRELATION ANALYSIS ==========
corr_matrix = df.corr()
target_corr = corr_matrix['target'].drop('target').sort_values(ascending=False)
print("Correlation with target:\n", target_corr)

# ========== 3. HANDLE MISSING VALUES ==========
# Check for missing
print(f"Missing values: {df.isnull().sum().sum()}")

# ========== 4. FILTER METHOD: Variance Threshold ==========
selector = VarianceThreshold(threshold=0.1)
X_high_variance = selector.fit_transform(X)
selected_features = X.columns[selector.get_support()]
print(f"Features after variance filter: {list(selected_features)}")

# ========== 5. FILTER METHOD: SelectKBest ==========
selector_kbest = SelectKBest(score_func=f_classif, k=2)
X_kbest = selector_kbest.fit_transform(X, y)
selected_kbest = X.columns[selector_kbest.get_support()]
print(f"Top 2 features: {list(selected_kbest)}")

# ========== 6. WRAPPER METHOD: RFE ==========
rfe = RFE(estimator=RandomForestClassifier(n_estimators=50, random_state=42), n_features_to_select=2)
rfe.fit(X, y)
print(f"RFE selected: {list(X.columns[rfe.support_])}")

# ========== 7. EMBEDDED METHOD: Feature Importance ==========
rf = RandomForestClassifier(n_estimators=100, random_state=42)
rf.fit(X, y)
importance = pd.DataFrame({
    'feature': X.columns,
    'importance': rf.feature_importances_
}).sort_values('importance', ascending=False)
print("Feature Importance:\n", importance)

# ========== 8. CONTROLLING INDICES & COLUMNS ==========
# Select top 2 features by importance
top_features = importance.head(2)['feature'].tolist()
df_selected = df[top_features + ['target']]
print(f"Selected dataset shape: {df_selected.shape}")
📊 Output: Correlation scores | Variance filter removed low-variance feature | RFE and Random Forest show consistent top features!

💡 Best Practice: Combine multiple feature selection methods (Filter → Wrapper → Embedded) for robust feature selection. Always scale features before methods sensitive to magnitude!
