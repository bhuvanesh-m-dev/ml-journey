📐 Key Formulas Cheat Sheet

Sigmoid Function

σ(z) = 1 / (1 + e⁻ᶻ)

Euclidean Distance

d(p,q) = √Σ(qᵢ - pᵢ)²

Entropy

H(S) = -Σpᵢ log₂(pᵢ)

SVM Margin

Margin = 2/||w||

Bayes' Theorem

P(A|B) = P(B|A)P(A) / P(B)

RMSE

RMSE = √(Σ(ŷᵢ - yᵢ)²/n)

R² Score

R² = 1 - MSE/Var(y)

F1-Score

F1 = 2·P·R / (P + R)

Eigen Decomposition

Σv = λv

K-Means Inertia

J = Σᵢ Σₓ∈Cᵢ ||x - μᵢ||²
