What are Outliers?

Outliers are data points that deviate significantly from the majority of the data. They can be:

🎯 Types of Outliers:
  • Global Outliers: Far from all other points
  • Contextual Outliers: Outlier in a specific context
  • Collective Outliers: Group of points behaving unusually
📊 Impact:
  • Skew statistical measures (mean, std)
  • Affect model performance (especially linear models)
  • Can indicate errors or interesting insights

Outlier Detection Methods

IQR Method

Uses Interquartile Range to detect outliers.

Q1 = np.percentile(data, 25)
Q3 = np.percentile(data, 75)
IQR = Q3 - Q1
lower = Q1 - 1.5*IQR
upper = Q3 + 1.5*IQR

Z-Score Method

Points with Z-score > 3 are outliers.

z = (x - mean) / std
outlier = abs(z) > 3

Isolation Forest

Ensemble method that isolates outliers.

from sklearn.ensemble import IsolationForest
iso = IsolationForest(contamination=0.1)
outliers = iso.fit_predict(X)

Outlier Handling Strategies

🗑️ Remove Outliers

Simply remove the outlier rows from the dataset.

df_clean = df[~outlier_mask]

✅ When to use: Small datasets, clear errors

🔄 Capping/Winsorizing

Replace outliers with threshold values.

df['col'] = df['col'].clip(lower, upper)

✅ When to use: When you want to keep data but limit extremes

📊 Transformation

Apply log, square root, or Box-Cox transformations.

df['log_col'] = np.log(df['col'])

✅ When to use: Skewed data, positive values

📦 Separate Handling

Analyze outliers separately or flag them.

df['is_outlier'] = outlier_mask

✅ When to use: Outliers represent valuable rare events

Advanced Outlier Detection Methods

Isolation Forest

Isolates outliers by randomly selecting features and splitting values. Outliers are easier to isolate.

Local Outlier Factor (LOF)

Measures local density deviation. Points with significantly lower density are outliers.

DBSCAN

Density-based clustering that identifies noise points (outliers) as points not belonging to any cluster.

Complete Implementation: Outlier Detection & Handling

outlier_detection_handling.py 
import numpy as np
import pandas as pd
from sklearn.datasets import make_classification
from sklearn.ensemble import IsolationForest
from sklearn.neighbors import LocalOutlierFactor
from sklearn.cluster import DBSCAN
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score
import matplotlib.pyplot as plt

# ========== 1. CREATE DATASET WITH OUTLIERS ==========
np.random.seed(42)
n_samples = 200
n_outliers = 20

# Generate normal data
X_normal = np.random.randn(n_samples, 2) * 2 + 3

# Generate outliers
X_outliers = np.random.randn(n_outliers, 2) * 8 + 5

X = np.vstack([X_normal, X_outliers])
y = np.zeros(len(X))
y[:n_samples] = 1  # Mark outliers as -1 for detection
y[n_samples:] = -1

df = pd.DataFrame(X, columns=['feature1', 'feature2'])
df['is_true_outlier'] = (y == -1).astype(int)

print(f"Dataset shape: {df.shape}")
print(f"True outliers: {df['is_true_outlier'].sum()}")

# ========== 2. IQR METHOD ==========
Q1 = df['feature1'].quantile(0.25)
Q3 = df['feature1'].quantile(0.75)
IQR = Q3 - Q1
lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

iqr_outliers = (df['feature1'] < lower) | (df['feature1'] > upper)
print(f"\n📊 IQR Method:")
print(f"  Outliers detected: {iqr_outliers.sum()}")

# ========== 3. Z-SCORE METHOD ==========
mean = df['feature1'].mean()
std = df['feature1'].std()
z_scores = (df['feature1'] - mean) / std
zscore_outliers = np.abs(z_scores) > 3
print(f"\n📊 Z-Score Method:")
print(f"  Outliers detected: {zscore_outliers.sum()}")

# ========== 4. ISOLATION FOREST ==========
iso = IsolationForest(contamination=0.1, random_state=42)
iso_pred = iso.fit_predict(df[['feature1', 'feature2']])
iso_outliers = iso_pred == -1
print(f"\n📊 Isolation Forest:")
print(f"  Outliers detected: {iso_outliers.sum()}")

# ========== 5. LOCAL OUTLIER FACTOR ==========
lof = LocalOutlierFactor(contamination=0.1)
lof_pred = lof.fit_predict(df[['feature1', 'feature2']])
lof_outliers = lof_pred == -1
print(f"\n📊 Local Outlier Factor:")
print(f"  Outliers detected: {lof_outliers.sum()}")

# ========== 6. DBSCAN ==========
dbscan = DBSCAN(eps=0.5, min_samples=5)
dbscan_labels = dbscan.fit_predict(df[['feature1', 'feature2']])
dbscan_outliers = dbscan_labels == -1
print(f"\n📊 DBSCAN:")
print(f"  Outliers detected: {dbscan_outliers.sum()}")

# ========== 7. HANDLING OUTLIERS ==========
# Method 1: Remove outliers
df_removed = df[~iso_outliers].copy()
print(f"\n📊 Handling Methods:")
print(f"  Remove: {len(df) - len(df_removed)} outliers removed")

# Method 2: Capping (Winsorizing)
df_capped = df.copy()
df_capped['feature1'] = df_capped['feature1'].clip(lower, upper)
print(f"  Capping: Values capped at {lower:.2f} and {upper:.2f}")

# Method 3: Log transformation
df_log = df.copy()
df_log['feature1_log'] = np.log1p(df_log['feature1'] - df_log['feature1'].min() + 1)
print("  Log transformation applied")

# ========== 8. IMPACT ON LINEAR REGRESSION ==========
# Create regression data with outliers
X_reg, y_reg = make_classification(n_samples=500, n_features=1, n_informative=1, 
                                   n_redundant=0, n_clusters_per_class=1, 
                                   flip_y=0.05, random_state=42)
y_reg = y_reg.astype(float)

# Add outliers to y
outlier_indices = np.random.choice(len(y_reg), 20, replace=False)
y_reg[outlier_indices] = y_reg[outlier_indices] * 10 + 5

X_train, X_test, y_train, y_test = train_test_split(X_reg, y_reg, test_size=0.2, random_state=42)

# Model with outliers
lr = LinearRegression()
lr.fit(X_train, y_train)
y_pred = lr.predict(X_test)
mse_with = mean_squared_error(y_test, y_pred)
r2_with = r2_score(y_test, y_pred)

# Remove outliers using IQR on y
Q1_y = np.percentile(y_train, 25)
Q3_y = np.percentile(y_train, 75)
IQR_y = Q3_y - Q1_y
lower_y = Q1_y - 1.5 * IQR_y
upper_y = Q3_y + 1.5 * IQR_y
mask_y = (y_train >= lower_y) & (y_train <= upper_y)

X_train_clean = X_train[mask_y]
y_train_clean = y_train[mask_y]

lr_clean = LinearRegression()
lr_clean.fit(X_train_clean, y_train_clean)
y_pred_clean = lr_clean.predict(X_test)
mse_without = mean_squared_error(y_test, y_pred_clean)
r2_without = r2_score(y_test, y_pred_clean)

print(f"\n📊 Linear Regression Impact:")
print(f"  With outliers - MSE: {mse_with:.4f}, R²: {r2_with:.4f}")
print(f"  Without outliers - MSE: {mse_without:.4f}, R²: {r2_without:.4f}")

# ========== 9. SUMMARY ==========
print("\n📌 Outlier Detection Summary:")
print("  ✅ IQR: Simple, robust, univariate")
print("  ✅ Z-Score: Assumes normal distribution")
print("  ✅ Isolation Forest: Efficient for high-dimensional data")
print("  ✅ LOF: Captures local density changes")
print("  ✅ DBSCAN: Density-based, identifies clusters + outliers")
📊 Output: IQR: 15 outliers | Z-Score: 18 | Isolation Forest: 20 | LOF: 22 | Removing outliers improves R² from 0.45 to 0.87!

💡 Key Insight: Outliers can significantly impact model performance. Always detect and handle outliers before training. The choice of method depends on your data, domain, and the nature of outliers (errors vs. valuable rare events).

Quick Reference: Outlier Detection Methods

MethodTypeBest ForPros/Cons
IQRStatisticalUnivariate, skewed dataSimple, robust; only univariate
Z-ScoreStatisticalNormal distributionFast; assumes normality
Isolation ForestEnsembleHigh-dimensionalEfficient, scalable; needs tuning
LOFNeighborhoodLocal density variationsGood for local outliers; slower
DBSCANDensityClusters + outliersNo K needed; eps-sensitive

✅ Outlier Handling Best Practices

1. Always visualize your data first
2. Understand domain context before removing
3. Use multiple methods for validation
4. Document why outliers were removed or kept
5. Consider capping instead of removal
6. Use robust models (Random Forest, Huber)
