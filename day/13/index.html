Deep GroupBy: Beyond Basic Aggregation

The GroupBy paradigm: Split → Apply → Combine. Master multiple aggregations, custom functions, and grouping on multiple keys.

import pandas as pd

df = pd.DataFrame({
    'dept': ['A','A','B','B','C','C'],
    'region': ['E','W','E','W','E','W'],
    'sales': [100,150,200,250,300,350]
})

# Multiple aggregations on multiple columns
result = df.groupby('dept').agg(
    total_sales=('sales', 'sum'),
    avg_sales=('sales', 'mean'),
    count=('sales', 'count')
)

# Group by multiple keys
result2 = df.groupby(['dept', 'region']).agg(
    total=('sales', 'sum'),
    std=('sales', 'std')
)

📊 Advanced GroupBy Methods:

.agg()Multiple aggregations at once
.transform()Broadcast results back to original shape
.filter()Keep groups based on condition
.apply()Custom function per group
.pipe()Chain group-specific operations

🎯 Pro Tip: Use pd.NamedAgg for explicit column naming in .agg(): .agg(total=pd.NamedAgg('sales', 'sum'))

Window Functions: Rolling, Expanding, EWM

Rolling Window

Moving calculations over fixed window size.

df['rolling_mean'] = df['value'].rolling(window=7).mean()

Expanding Window

Cumulative calculations from start.

df['cumsum'] = df['value'].expanding().sum()

EWM (Exponential Weighted)

More weight to recent observations.

df['ewm'] = df['value'].ewm(span=30).mean()

📈 Financial Applications: Rolling volatility (std), expanding Sharpe ratio, EWM for trend following strategies.

Merge Strategies: Combining DataFrames

Inner Join
Only matching keys
Left Join
All left keys
Right Join
All right keys
Outer Join
All keys from both
# Different join types
pd.merge(df1, df2, on='key', how='inner')
pd.merge(df1, df2, on='key', how='left')
pd.merge(df1, df2, on='key', how='right')
pd.merge(df1, df2, on='key', how='outer')

# Merge on multiple keys
pd.merge(df1, df2, on=['key1', 'key2'])

# Merge with different column names
pd.merge(df1, df2, left_on='id1', right_on='id2')

# Concatenation (vertical stacking)
pd.concat([df1, df2], axis=0, ignore_index=True)

# Join using index
df1.join(df2, how='left')

⚡ Performance Tips for Merging:

  • Sort keys before merging with sort=True
  • Use indicator=True to track merge sources
  • For large DataFrames, consider dask.dataframe
  • Set indexes on merge keys for faster joins

Transform & Custom Apply: Broadcasting Power

🔄 Transform: Group Results Back to Original Shape

# Add group mean to each row
df['dept_mean'] = df.groupby('dept')['sales'].transform('mean')

# Normalize within group (z-score)
df['z_score'] = df.groupby('dept')['sales'].transform(
    lambda x: (x - x.mean()) / x.std()
)

# Rank within group
df['rank'] = df.groupby('dept')['sales'].transform('rank', ascending=False)

📊 Custom Apply: Your Own Logic

# Custom aggregation per group
def custom_agg(group):
    return pd.Series({
        'total': group['sales'].sum(),
        'range': group['sales'].max() - group['sales'].min(),
        'cv': group['sales'].std() / group['sales'].mean()
    })
    
result = df.groupby('dept').apply(custom_agg)

# Row-wise apply (use carefully!)
df['category'] = df.apply(
    lambda row: 'High' if row['sales'] > 200 else 'Low', axis=1
)

⚡ transform() preserves index, apply() can change shape — use transform when possible for speed!

Memory Optimization: Handling Large Datasets

# 1. Downcast numeric types
for col in df.select_dtypes('int').columns:
    df[col] = pd.to_numeric(df[col], downcast='integer')
    
for col in df.select_dtypes('float').columns:
    df[col] = pd.to_numeric(df[col], downcast='float')

# 2. Convert object to categorical
for col in df.select_dtypes('object').columns:
    if df[col].nunique() / len(df) < 0.5:  # <50% unique
        df[col] = df[col].astype('category')

# 3. Read only needed columns
df = pd.read_csv('large.csv', usecols=['col1', 'col2'])

# 4. Chunk reading for huge files
chunks = pd.read_csv('huge.csv', chunksize=10000)

Method Chaining & Pipelines: Clean, Readable Code

Method chaining creates readable, maintainable data transformation pipelines.

# Without chaining (messy)
df = pd.read_csv('data.csv')
df = df.dropna()
df = df[df['age'] > 18]
df['salary_k'] = df['salary'] / 1000
result = df.groupby('dept').mean()

# With chaining (clean!)
result = (pd.read_csv('data.csv')
    .dropna()
    .query('age > 18')
    .assign(salary_k=lambda x: x['salary'] / 1000)
    .groupby('dept')
    .mean()
)

# Using pipe for custom functions
def remove_outliers(df, col, threshold=3):
    z_scores = (df[col] - df[col].mean()) / df[col].std()
    return df[abs(z_scores) < threshold]

clean_df = (df
    .pipe(remove_outliers, 'salary')
    .pipe(remove_outliers, 'age')
)

🔗 Pipeline Best Practices:

  • Use .assign() for creating new columns
  • Use .query() for filtering (cleaner syntax)
  • Use .pipe() for custom reusable functions
  • Wrap pipeline in parentheses for line breaks
  • Comment each step for clarity

import pandas as pd
import numpy as np

# ========== CREATE SAMPLE DATA ==========
np.random.seed(42)
df = pd.DataFrame({
    'date': pd.date_range('2024-01-01', periods=1000),
    'region': np.random.choice(['North', 'South', 'East', 'West'], 1000),
    'product': np.random.choice(['A', 'B', 'C'], 1000),
    'sales': np.random.normal(1000, 200, 1000).astype(int),
    'units': np.random.poisson(50, 1000)
})

# ========== 1. ADVANCED GROUPBY ==========
result = (df.groupby(['region', 'product'])
    .agg(
        total_sales=('sales', 'sum'),
        avg_units=('units', 'mean'),
        transaction_count=('sales', 'count'),
        sales_std=('sales', 'std')
    ))

# ========== 2. WINDOW FUNCTIONS ==========
df['sales_7d_ma'] = df['sales'].rolling(window=7).mean()
df['sales_cumsum'] = df['sales'].expanding().sum()
df['sales_ewm'] = df['sales'].ewm(span=30).mean()

# ========== 3. TRANSFORM (Group Normalization) ==========
df['region_normalized'] = df.groupby('region')['sales'].transform(
    lambda x: (x - x.mean()) / x.std()
)

# ========== 4. METHOD CHAINING PIPELINE ==========
clean_data = (df
    .assign(date=lambda x: pd.to_datetime(x['date']))
    .query('sales > 500 and units > 10')
    .dropna()
    .assign(profit=lambda x: x['sales'] * 0.25)
    .groupby(['region', pd.Grouper(key='date', freq='M')])
    .agg(total_profit=('profit', 'sum'))
)

print(f"Shape: {clean_data.shape}")
print(f"Memory: {df.memory_usage(deep=True).sum() / 1024**2:.2f} MB")
