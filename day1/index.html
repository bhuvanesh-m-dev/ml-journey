The Data Trio

Data Science

The field of extracting meaningful insights and knowledge from data using statistics, programming, and machine learning techniques.

Goals

  • Collect data
  • Clean and process data
  • Analyze patterns
  • Build predictive models
  • Support decision-making
Example:
A company analyzes customer purchase data to understand buying behavior and improve sales.

Data Analysis

The process of inspecting, cleaning, transforming, and interpreting data to discover useful information.

Steps

  1. Collect data
  2. Clean data
  3. Explore data
  4. Find patterns
  5. Draw conclusions
Example:
Analyzing monthly sales data to identify the best-performing products.

Data Analytics

Focuses on examining datasets to answer specific business questions and support decision-making.

Types

1. Descriptive: "What happened?" (Reports)
2. Diagnostic: "Why did it happen?" (Investigate)
3. Predictive: "What will happen?" (Predict)
4. Prescriptive: "What should be done?" (Recommend)

Types of Machine Learning

Supervised Learning

The model learns from labeled data. Input data is provided along with the correct output.

A. Classification

Used when the output belongs to categories (discrete).

Email Spam
Email Not Spam

Examples: Spam/Not Spam, Cat/Dog, Pass/Fail

B. Regression

Used when the output is a continuous numerical value.

House Size (small) ₹50,00,000
House Size (large) ₹75,00,000

Examples: Price, Temperature, Stock Prediction

Unsupervised Learning

The model learns from unlabeled data. The algorithm discovers hidden patterns without knowing the correct answers.

A. Clustering

Grouping similar data points together.

Customer Data Group A
Customer Data Group B

Examples: Customer Segmentation, Recommendation Systems

B. Dimensionality Reduction

Reducing the number of features while preserving important information.

100 Features 10 Important Features

Benefits: Faster training, Reduced storage, Better visualization

Semi-Supervised Learning

Uses a small amount of labeled data and a large amount of unlabeled data.
