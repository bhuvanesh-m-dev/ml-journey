🔬 Advanced Array Operations

📊 View vs Copy: 
# View (shares data)
arr = np.array([1,2,3,4,5])
view = arr[1:4]
view[0] = 99  # Changes original!

# Copy (independent)
copy = arr[1:4].copy()
copy[0] = 99  # Original unchanged
🔄 Fancy Indexing: 
arr = np.array([10,20,30,40,50])
indices = [0, 2, 4]
result = arr[indices]  # [10, 30, 50]

# Boolean indexing
mask = arr > 30
result = arr[mask]  # [40, 50]
📐 Advanced Slicing: 
arr = np.arange(24).reshape(2,3,4)
# Step slicing
slice_1 = arr[:, :, ::2]  # Every 2nd element in last dim
# Ellipsis (...) for all dims
slice_2 = arr[..., 0]  # All rows, first column

🧠 Memory Layout: C-order vs F-order

NumPy uses C-order (row-major) by default, but supports F-order (column-major) for compatibility with Fortran.

📊 C-Order (Row-Major): 
arr = np.array([[1,2,3],[4,5,6]], order='C')
# Memory: [1,2,3,4,5,6]
# Rows are contiguous
📊 F-Order (Column-Major): 
arr = np.array([[1,2,3],[4,5,6]], order='F')
# Memory: [1,4,2,5,3,6]
# Columns are contiguous
🔄 Changing Order: 
c_arr = np.array([[1,2],[3,4]])
f_arr = np.asfortranarray(c_arr)  # Convert to F-order
c_arr2 = np.ascontiguousarray(f_arr)  # Convert to C-order

🌊 Broadcasting: Deep Understanding

Broadcasting allows operations between arrays of different shapes by "stretching" smaller arrays.

📐 Rules:
  • Dimensions are compared from trailing (rightmost) side
  • Compatible if dimensions are equal or one is 1
  • Missing dimensions are treated as 1
🔍 Examples: 
# 2D + 1D
A = np.ones((3,4))  # shape (3,4)
B = np.array([1,2,3,4])  # shape (4,)
# B broadcasts to (3,4)
C = A + B

# 2D + scalar
A = np.ones((3,4))
C = A + 5  # 5 broadcasts to (3,4)

⚡ Universal Functions (ufuncs)

ufuncs are vectorized functions that operate element-wise on arrays. They are fast because they're implemented in C.

📊 Unary ufuncs: 
np.sqrt(arr)      # Square root
np.exp(arr)       # Exponential
np.log(arr)       # Natural log
np.sin(arr)       # Sine
np.abs(arr)       # Absolute value
np.negative(arr)  # Negate
📊 Binary ufuncs: 
np.add(a, b)      # a + b
np.subtract(a,b)  # a - b
np.multiply(a,b)  # a * b
np.divide(a,b)    # a / b
np.power(a,b)     # a^b
np.maximum(a,b)   # Element-wise max
🔄 Reduce & Accumulate: 
np.add.reduce(arr)  # Sum of all elements
np.multiply.accumulate(arr)  # Cumulative product
np.add.at(arr, idx, val)  # In-place addition

💡 ufuncs are the key to NumPy's performance — always use them over Python loops!

⚡ Performance Optimization Techniques

🚀 Avoid Python Loops: 
# SLOW
for i in range(len(arr)):
    arr[i] = arr[i] * 2

# FAST (vectorized)
arr = arr * 2
💾 Use In-place Operations: 
np.multiply(arr, 2, out=arr)  # In-place multiply
np.add(arr, 10, out=arr)       # In-place add
📊 Use NumPy Functions: 
# Instead of sum(arr) / len(arr)
np.mean(arr)

# Instead of manual dot product
np.dot(a, b)

📊 Optimization Checklist

  • ✅ Use vectorized operations (50-100x faster)
  • ✅ Use in-place operations (save memory)
  • ✅ Use ufuncs instead of Python functions
  • ✅ Use views instead of copies when possible
  • ✅ Use np.einsum for complex tensor operations
  • ✅ Use Numba or Cython for extreme performance

💡 Rule: If you're using a Python loop on a NumPy array, you're probably doing it wrong!
