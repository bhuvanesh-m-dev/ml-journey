Why Cross-Validation?

Cross-Validation (CV) is a resampling technique used to evaluate machine learning models on a limited data sample.

🎯 Why use CV?
  • Better estimate of model performance
  • Reduces overfitting
  • Maximizes use of available data
  • Helps with hyperparameter tuning
📊 Single Split vs CV 
# Single Split (high variance)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Cross-Validation (more robust)
scores = cross_val_score(model, X, y, cv=5)

🔄 K-Fold Cross-Validation

K-Fold CV splits data into K equal-sized folds. Each fold serves as test set once, while the remaining K-1 folds are used for training.

from sklearn.model_selection import KFold, cross_val_score

kf = KFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(model, X, y, cv=kf, scoring='accuracy')

print(f"CV Scores: {scores}")
print(f"Mean: {scores.mean():.4f} ± {scores.std():.4f}")

📌 K-Fold Rules:

  • K=5 or K=10 are most common
  • Larger K = less bias, more variance
  • Always shuffle data before splitting

📊 Stratified K-Fold

Stratified K-Fold preserves the class distribution in each fold. Essential for imbalanced datasets.

from sklearn.model_selection import StratifiedKFold

skf = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(model, X, y, cv=skf)

# Check class distribution per fold
for train_idx, test_idx in skf.split(X, y):
    print(f"Train: {np.bincount(y[train_idx])}")
    print(f"Test: {np.bincount(y[test_idx])}")

When to use: Classification problems with imbalanced classes. Always use stratified for classification!

🔬 Advanced Cross-Validation Methods

Leave-One-Out (LOOCV)

K = n_samples. Each sample is test set once.

from sklearn.model_selection import LeaveOneOut

✅ Best for small datasets

❌ Very expensive

Repeated K-Fold

Run K-Fold multiple times with different shuffles.

from sklearn.model_selection import RepeatedKFold

✅ More stable estimates

⚠️ Computationally heavier

Time Series Split

Preserves temporal order (no shuffling).

from sklearn.model_selection import TimeSeriesSplit

✅ For time series data

❌ Cannot shuffle

Group K-Fold

Ensures groups are kept together.

from sklearn.model_selection import GroupKFold

✅ For grouped data (patients, users)

ShuffleSplit

Random train-test splits repeated n times.

from sklearn.model_selection import ShuffleSplit

✅ Flexible, good for large data

StratifiedShuffleSplit

ShuffleSplit with stratification.

from sklearn.model_selection import StratifiedShuffleSplit

✅ Best of both worlds

📝 Complete Implementation: All CV Methods

cross_validation_complete.py 
import numpy as np
import pandas as pd
from sklearn.datasets import load_breast_cancer, make_classification
from sklearn.model_selection import (
    KFold, StratifiedKFold, LeaveOneOut, LeavePOut,
    RepeatedKFold, ShuffleSplit, StratifiedShuffleSplit,
    TimeSeriesSplit, GroupKFold, GroupShuffleSplit,
    cross_val_score, cross_validate, cross_val_predict,
    GridSearchCV, RandomizedSearchCV
)
from sklearn.ensemble import RandomForestClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score, f1_score, roc_auc_score
from sklearn.preprocessing import StandardScaler
import time

print("🔄 CROSS-VALIDATION COMPLETE GUIDE")
print("=" * 50)

# Load data
data = load_breast_cancer()
X, y = data.data, data.target
print(f"\nDataset: {X.shape[0]} samples, {X.shape[1]} features")
print(f"Class distribution: {np.bincount(y)}")

# ========== 1. K-FOLD ==========
print("\n📊 1. K-Fold CV:")
kf = KFold(n_splits=5, shuffle=True, random_state=42)
model = RandomForestClassifier(n_estimators=50, random_state=42)
scores_kf = cross_val_score(model, X, y, cv=kf, scoring='accuracy')
print(f"  K-Fold: {scores_kf.mean():.4f} ± {scores_kf.std():.4f}")

# ========== 2. STRATIFIED K-FOLD ==========
print("\n📊 2. Stratified K-Fold:")
skf = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
scores_skf = cross_val_score(model, X, y, cv=skf, scoring='accuracy')
print(f"  Stratified: {scores_skf.mean():.4f} ± {scores_skf.std():.4f}")

# ========== 3. LEAVE-ONE-OUT (LOOCV) ==========
print("\n📊 3. Leave-One-Out (LOOCV):")
# Only on subset for speed
X_subset = X[:50]
y_subset = y[:50]
loo = LeaveOneOut()
scores_loo = cross_val_score(model, X_subset, y_subset, cv=loo, scoring='accuracy')
print(f"  LOOCV: {scores_loo.mean():.4f} ± {scores_loo.std():.4f}")

# ========== 4. REPEATED K-FOLD ==========
print("\n📊 4. Repeated K-Fold:")
rkf = RepeatedKFold(n_splits=5, n_repeats=3, random_state=42)
scores_rkf = cross_val_score(model, X, y, cv=rkf, scoring='accuracy')
print(f"  Repeated K-Fold: {scores_rkf.mean():.4f} ± {scores_rkf.std():.4f}")

# ========== 5. SHUFFLE SPLIT ==========
print("\n📊 5. Shuffle Split:")
ss = ShuffleSplit(n_splits=10, test_size=0.2, random_state=42)
scores_ss = cross_val_score(model, X, y, cv=ss, scoring='accuracy')
print(f"  ShuffleSplit: {scores_ss.mean():.4f} ± {scores_ss.std():.4f}")

# ========== 6. STRATIFIED SHUFFLE SPLIT ==========
print("\n📊 6. Stratified Shuffle Split:")
sss = StratifiedShuffleSplit(n_splits=10, test_size=0.2, random_state=42)
scores_sss = cross_val_score(model, X, y, cv=sss, scoring='accuracy')
print(f"  StratifiedShuffleSplit: {scores_sss.mean():.4f} ± {scores_sss.std():.4f}")

# ========== 7. TIME SERIES SPLIT ==========
print("\n📊 7. Time Series Split:")
tscv = TimeSeriesSplit(n_splits=5)
X_ts = np.arange(100).reshape(-1, 1)
y_ts = np.arange(100)
for train_idx, test_idx in tscv.split(X_ts):
    print(f"  Train: {len(train_idx)}, Test: {len(test_idx)}")

# ========== 8. CROSS_VALIDATE (Multiple Metrics) ==========
print("\n📊 8. Cross-Validate with Multiple Metrics:")
scoring = ['accuracy', 'f1_macro', 'roc_auc_ovr']
cv_results = cross_validate(model, X, y, cv=5, scoring=scoring, return_train_score=True)

for metric in scoring:
    test_key = f'test_{metric}'
    train_key = f'train_{metric}'
    print(f"  {metric}: {cv_results[test_key].mean():.4f} ± {cv_results[test_key].std():.4f}")

# ========== 9. CROSS_VAL_PREDICT ==========
print("\n📊 9. Cross-Validation Predictions:")
y_pred_cv = cross_val_predict(model, X, y, cv=5)
acc_cv_pred = accuracy_score(y, y_pred_cv)
print(f"  CV Prediction Accuracy: {acc_cv_pred:.4f}")

# ========== 10. GRID SEARCH WITH CV ==========
print("\n📊 10. GridSearchCV:")
param_grid = {
    'n_estimators': [50, 100],
    'max_depth': [5, 10, None],
    'min_samples_split': [2, 5]
}
grid = GridSearchCV(RandomForestClassifier(random_state=42), param_grid, cv=5, scoring='accuracy')
grid.fit(X, y)
print(f"  Best params: {grid.best_params_}")
print(f"  Best score: {grid.best_score_:.4f}")

# ========== 11. RANDOMIZED SEARCH ==========
print("\n📊 11. RandomizedSearchCV:")
from sklearn.model_selection import RandomizedSearchCV
random_search = RandomizedSearchCV(
    RandomForestClassifier(random_state=42),
    param_distributions=param_grid,
    n_iter=10,
    cv=5,
    random_state=42
)
random_search.fit(X, y)
print(f"  Best params: {random_search.best_params_}")
print(f"  Best score: {random_search.best_score_:.4f}")

# ========== 12. COMPARISON TABLE ==========
print("\n📊 12. Method Comparison:")
methods = {
    'KFold': scores_kf.mean(),
    'StratifiedKFold': scores_skf.mean(),
    'RepeatedKFold': scores_rkf.mean(),
    'ShuffleSplit': scores_ss.mean(),
    'StratifiedShuffleSplit': scores_sss.mean()
}
for name, score in sorted(methods.items(), key=lambda x: x[1], reverse=True):
    print(f"  {name}: {score:.4f}")

print("\n📌 Cross-Validation Summary:")
print("  ✅ K-Fold: Standard choice")
print("  ✅ Stratified: For classification (imbalanced)")
print("  ✅ LOOCV: For small datasets")
print("  ✅ Repeated K-Fold: More stable estimates")
print("  ✅ TimeSeriesSplit: For temporal data")
print("  ✅ GroupKFold: For grouped data")
📊 Output: K-Fold: 0.965 | Stratified: 0.965 | GridSearchCV best score: 0.971 | Cross-validation provides robust model evaluation!

💡 Key Insight: Cross-validation is essential for reliable model evaluation. Always use Stratified K-Fold for classification problems. For hyperparameter tuning, use cross-validation within GridSearchCV or RandomizedSearchCV to avoid overfitting.

✅ Cross-Validation Best Practices

1. Always use Stratified K-Fold for classification
2. Choose K=5 or K=10 (standard)
3. Shuffle data before splitting
4. Use TimeSeriesSplit for temporal data
5. Use GroupKFold for grouped data
6. Nested CV for hyperparameter tuning
7. Report mean ± std of CV scores
8. Use LOOCV only for small datasets
