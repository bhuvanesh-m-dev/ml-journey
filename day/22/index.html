What is Clustering?

Clustering is an unsupervised learning technique that groups similar data points together. It finds hidden patterns without predefined labels.

🎯 Goal:

Group data points so that points in the same cluster are more similar to each other than to points in other clusters.

📊 Real-World Applications:
  • Customer Segmentation (marketing)
  • Image Segmentation (computer vision)
  • Anomaly Detection (fraud detection)
  • Document Clustering (NLP)
  • Recommender Systems

📊 Types of Clustering

● Partitioning: K-Means, K-Medoids
● Hierarchical: Agglomerative, Divisive
● Density-Based: DBSCAN, OPTICS
● Model-Based: Gaussian Mixture Models

💡 K-Means is the most popular and widely used clustering algorithm!

K-Means Clustering: Core Concept

K-Means partitions data into K clusters. Each cluster is represented by its centroid (mean of all points in that cluster).

from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=3, random_state=42)
labels = kmeans.fit_predict(X)
centroids = kmeans.cluster_centers_

Objective: Minimize within-cluster sum of squares (inertia)

\[ J = \sum_{i=1}^{K} \sum_{x \in C_i} ||x - \mu_i||^2 \]

K-Means Algorithm: Step by Step

🔄 The Algorithm:

  1. 1. Initialize K centroids randomly (or with K-Means++)
  2. 2. Assign each point to the nearest centroid (Euclidean distance)
  3. 3. Recalculate centroids as the mean of assigned points
  4. 4. Repeat steps 2-3 until convergence (centroids stop moving)

📌 Key Concepts:

Cluster:Group of similar points
Centroid:Mean of points in cluster
Distance:Euclidean distance (default)
Inertia:Within-cluster sum of squares
Convergence:When inertia stops decreasing

⚡ Time Complexity: O(K × n × d × iterations) — K clusters, n samples, d dimensions, iterations until convergence

Choosing K: Elbow Method & Silhouette Score

The Elbow Method plots inertia for different K values. The "elbow" point is the optimal K.

from sklearn.cluster import KMeans

inertias = []
K_range = range(1, 11)
for k in K_range:
    kmeans = KMeans(n_clusters=k)
    kmeans.fit(X)
    inertias.append(kmeans.inertia_)

# Elbow plot
plt.plot(K_range, inertias, 'bo-')
plt.xlabel('K')
plt.ylabel('Inertia')

Elbow Point: Where inertia starts decreasing slowly

Alternative: Silhouette Score (higher = better clustering)

🚀 K-Means++: Better Initialization

K-Means++ improves the initialization step by spreading centroids far apart, leading to better and more consistent clustering.

Step 1: Choose first centroid randomly
Step 2: Select next centroid with probability proportional to distance²
Step 3: Repeat until K centroids chosen
✓ Faster convergence, better results!

Complete Implementation: K-Means Clustering

kmeans_clustering_demo.py 
import numpy as np
import pandas as pd
from sklearn.datasets import make_blobs, make_moons, load_digits
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score, silhouette_samples, calinski_harabasz_score, davies_bouldin_score
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt

# ========== 1. CREATE DATASET ==========
X, y_true = make_blobs(n_samples=500, centers=4, cluster_std=0.8, random_state=42)
print(f"Dataset: {X.shape[0]} samples, {X.shape[1]} features, {len(np.unique(y_true))} true clusters")

# ========== 2. SCALE DATA ==========
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# ========== 3. FIND OPTIMAL K (ELBOW METHOD) ==========
inertias = []
silhouette_scores = []
K_range = range(2, 11)

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X_scaled)
    inertias.append(kmeans.inertia_)
    silhouette_scores.append(silhouette_score(X_scaled, kmeans.labels_))

print("\n📊 K Selection:")
for i, k in enumerate(K_range):
    print(f"  K={k}: Inertia={inertias[i]:.0f}, Silhouette={silhouette_scores[i]:.4f}")

best_k = K_range[np.argmax(silhouette_scores)]
print(f"\n✅ Optimal K (best silhouette): {best_k}")

# ========== 4. TRAIN FINAL MODEL ==========
kmeans_final = KMeans(n_clusters=best_k, random_state=42, n_init=10)
labels = kmeans_final.fit_predict(X_scaled)
centroids = kmeans_final.cluster_centers_

print(f"\n📊 Final Clustering:")
print(f"  Number of clusters: {best_k}")
print(f"  Inertia: {kmeans_final.inertia_:.2f}")
print(f"  Silhouette Score: {silhouette_score(X_scaled, labels):.4f}")

# ========== 5. EVALUATION METRICS ==========
print("\n📊 Clustering Evaluation Metrics:")
print(f"  Silhouette Score: {silhouette_score(X_scaled, labels):.4f}")
print(f"  Calinski-Harabasz Score: {calinski_harabasz_score(X_scaled, labels):.2f}")
print(f"  Davies-Bouldin Score: {davies_bouldin_score(X_scaled, labels):.4f}")

# ========== 6. NON-LINEAR DATA (MOONS) ==========
X_moons, _ = make_moons(n_samples=300, noise=0.1, random_state=42)
X_moons_scaled = StandardScaler().fit_transform(X_moons)

kmeans_moons = KMeans(n_clusters=2, random_state=42, n_init=10)
labels_moons = kmeans_moons.fit_predict(X_moons_scaled)

print(f"\n📊 Moons Dataset:")
print(f"  Silhouette: {silhouette_score(X_moons_scaled, labels_moons):.4f}")

# ========== 7. INITIALIZATION METHODS ==========
print("\n📊 Initialization Comparison:")

# Random initialization
kmeans_random = KMeans(n_clusters=3, init='random', random_state=42, n_init=1)
kmeans_random.fit(X_scaled)

# K-Means++ initialization
kmeans_plus = KMeans(n_clusters=3, init='k-means++', random_state=42, n_init=1)
kmeans_plus.fit(X_scaled)

print(f"  Random init inertia: {kmeans_random.inertia_:.2f}")
print(f"  K-Means++ init inertia: {kmeans_plus.inertia_:.2f}")

# ========== 8. PREDICT NEW DATA ==========
new_points = np.array([[0, 0], [3, 3], [-2, -2]])
new_points_scaled = scaler.transform(new_points)
predictions = kmeans_final.predict(new_points_scaled)

print(f"\n📊 New point predictions: {predictions}")

# ========== 9. CLUSTER ANALYSIS ==========
print("\n📊 Cluster Analysis:")
for i in range(best_k):
    cluster_points = X_scaled[labels == i]
    print(f"  Cluster {i}: {len(cluster_points)} points, " +
          f"mean distance to centroid: {np.mean(np.linalg.norm(cluster_points - centroids[i], axis=1)):.4f}")

# ========== 10. ADVANCED: K-MEANS FOR IMAGE COMPRESSION ==========
digits = load_digits()
X_digits = digits.data

kmeans_compress = KMeans(n_clusters=64, random_state=42, n_init=10)
kmeans_compress.fit(X_digits)

print(f"\n📊 Image Compression (digits):")
print(f"  Compressed {X_digits.shape[0]} images into {kmeans_compress.cluster_centers_.shape[0]} centroids")

# ========== SUMMARY ==========
print("\n📌 K-Means Key Takeaways:")
print("  ✅ Pros: Simple, fast, scalable, interpretable")
print("  ❌ Cons: Sensitive to initialization, assumes spherical clusters")
print("  💡 Use Elbow Method + Silhouette Score to choose K")
print("  🔄 Scale data before clustering (StandardScaler)")
📊 Output: Optimal K=4, Silhouette=0.67, Inertia=148.32 | K-Means++ gives better initialization (lower inertia)

💡 Key Insight: K-Means is scalable and works well for spherical clusters. Always scale features before clustering. Use Elbow Method and Silhouette Score to determine optimal K. For non-spherical clusters, consider DBSCAN or Gaussian Mixture Models.

K-Means: Strengths & Limitations

✅ Advantages

  • Simple, intuitive, and easy to implement
  • Fast and scalable (O(K·n·d·iterations))
  • Works well with large datasets
  • Guaranteed to converge (may be local optimum)
  • Interpretable results (centroids are meaningful)

❌ Limitations

  • Requires specifying K (number of clusters)
  • Sensitive to initialization (use K-Means++)
  • Assumes spherical clusters of similar size
  • Sensitive to outliers
  • Struggles with non-linearly separable data
