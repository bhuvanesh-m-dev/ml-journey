History of NumPy: From Numeric to NumPy

Who Created NumPy?

  • 👤 Travis Oliphant - Primary creator, merged Numeric and Numarray
  • 👤 Jim Hugunin - Created original Numeric (1995)
  • 👤 Paul Dubois - Maintained Numeric at LLNL
  • 👤 Charles R. Harris, Eric Jones, Robert Kern - Major contributors

Why Was NumPy Created?

  • ◆ Python's native lists were too slow for numerical computing
  • ◆ Need for efficient multi-dimensional array operations
  • ◆ Two competing libraries: Numeric (fast) and Numarray (flexible)
  • ◆ Travis Oliphant merged them into NumPy in 2005
  • ◆ Foundation for SciPy, Pandas, Matplotlib, scikit-learn

📅 NumPy Evolution Timeline

1995: Jim Hugunin creates Numeric at MIT
2001: Numarray developed as alternative with more features
2005: Travis Oliphant merges Numeric + Numarray → NumPy 1.0
2006: NumPy becomes foundation of SciPy stack
2011: Pandas built on top of NumPy
Today: 15+ million downloads/month, core of data science ecosystem

ndarray: N-Dimensional Array

The ndarray is NumPy's core data structure — a homogeneous, fixed-size, multi-dimensional container for numerical data.

import numpy as np

arr = np.array([1, 2, 3, 4, 5])

# 1D array (vector)

matrix = np.array([[1, 2, 3], [4, 5, 6]])

# 2D array (matrix) - shape (2, 3)

Vectorization: Python Loops vs NumPy

🐍 Pure Python Loop

arr = list(range(1000000))
result = []
for x in arr:
    result.append(x * 2)

⏱️ ~50-100 ms

🚀 NumPy Vectorized

import numpy as np
arr = np.arange(1000000)
result = arr * 2

⏱️ ~1-2 ms (50x faster!)

⚡ Why so fast? NumPy operations run in compiled C code, bypass Python's interpreter overhead. Plus SIMD vectorization and optimized memory access.

Broadcasting: Operations on Different Shapes

Broadcasting allows arithmetic operations between arrays of different shapes by "stretching" smaller arrays to match larger ones.

Add scalar to array:

[1, 2, 3] + 10 = [11, 12, 13]

Add row vector to matrix:

[[1,2],[3,4]] + [10,20] = [[11,22],[13,24]]

Broadcasting Rules:

  • Arrays are compatible if dimensions are equal or one is 1
  • Missing dimensions are treated as 1
  • Example: (3,4) + (4,) → (3,4) + (1,4) → works!

NumPy in Action

numpy_demo.py 
import numpy as np

# 1. Create arrays
arr1d = np.array([1, 2, 3, 4, 5])
arr2d = np.array([[1, 2, 3], [4, 5, 6]])
zeros = np.zeros((3, 4))
ones = np.ones((2, 3))
eye = np.eye(4)  # Identity matrix
range_arr = np.arange(0, 20, 2)  # [0,2,4,...,18]
lin = np.linspace(0, 1, 10)  # 10 points from 0 to 1
random_arr = np.random.randn(100)  # Normal distribution

# 2. Array attributes
print(f"Shape: {arr2d.shape}, Dims: {arr2d.ndim}, Size: {arr2d.size}")

# 3. Vectorized operations
arr = np.arange(1, 1000000)
result = np.sqrt(arr) + np.sin(arr) * 2  # Lightning fast!

# 4. Indexing & Slicing
matrix = np.random.randint(0, 100, (5, 5))
col_2 = matrix[:, 1]  # Second column
submatrix = matrix[1:4, 1:4]  # 3x3 submatrix
bool_mask = matrix > 50
filtered = matrix[bool_mask]  # Elements > 50

# 5. Aggregations
print(f"Mean: {np.mean(matrix):.2f}, Std: {np.std(matrix):.2f}")
print(f"Row sums: {np.sum(matrix, axis=1)}")
📊 Output: Shape: (2,3), Dims: 2, Size: 6 | Mean: 49.52, Std: 28.91 | NumPy enables fast numerical computing!
